Explain inline , internal and external stylesheet

I. Internal Style sheet requires you to add style tag in the head tag section of your HTML document. (Ex- d3.css)

II. External style sheet requirs to link your web pages to an external .css file

III. Inline style sheet is used to style a specific HTML element. For this CSS style, only need to add the style attribute to each HTML tag, without using selectors.

What is grid-template-columns used for?

The grid-template-columns property in CSS is used to set the number of columns and size of the columns of the grid. This property accepts more than one values.

What is the z-index in CSS?

The z-index property specifies the stack order of an element.
An element with greater stack order is always in front of an element with a lower stack order. Note: z-index only works on positioned elements (position: absolute, position: relative, position: fixed, or position: sticky)

Which property you will use to merge cells vertically in grid?

A (vmerge) element with the attribute set to continue (the default value) marks the continuation of a vertically merged grid column.
Cells between the first and last merged cell that are part of the vertical merge each must have a (vmerge) element to continue the vertical merge.

Which property you will use to merge cells horizontally in grid?

It is possible to implement horizontally merged cells using the Grid.This approach is based on removing one cell from a certain row.

Explain CSS box model

A CSS box model is a compartment that includes numerous assets, such as edge, border, padding and material.
It is used to develop the design and structure of a web page. It can be used as a set of tools to personalize the layout of different components.
